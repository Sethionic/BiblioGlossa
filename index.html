BiblioGlossa

This repository hosts the materials for a Greek language learning aid. The name comes from the words γλωσσα ("Language") and βιλιον ("Book")

Technical Details to Date

The project is intended to use HTML5 to store a local SQLite3 database and makes calls from modules to it.

There are 3 modules planned so far:

Flashcard: Display an interface similar to flashcards that the user can filter by whatever constraints they desire the words that will randomly appear.

Lexicon: Allow the user to get a listing of all words within the database that fit their search constraints

Parsing: An experimental engine that will attempt to generate Greek words based on parameters specified by the user
