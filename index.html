BiblioGlossa

This repository hosts the materials for a Greek language learning aid. The name comes from the words γλωσσα ("Language") and βιλιον ("Book")

Technical Details to Date

The project is intended to use HTML5 to store a local database and makes calls from modules to it.

There are 3 modules planned so far:

Flashcard: Display an interface similar to flashcards that the user can filter by whatever constraints they desire the words that will randomly appear. By default, there will be a main deck containing all words as well as a deck containing the words for each chapter. New decks can be created using the Lexicon page.

Lexicon: Allow the user to get a listing of all words within the database that fit their search constraints and create new flashcard decks from their results

Parsing: An experimental engine that will attempt to generate Greek words based on parameters specified by the user. [Outside of current scope]
